### **Description**
By default if you press E you'll go from Gear to Health to Skills and so on, once you reach the Prestige tab if you press E once more you'll cycle back to the first (Overall) tab. Pressing Q will do the same thing but in the reverse order. \
This works both in the main menu as well as in the in raid inventory. In the main menu the Search function in your stash has priority over the hotkeys, so if you want to search for something in your stash you won't be accidentally changing tabs as you type. \
You can change the hotkeys to any key you'd like within the F12 menu > desze-Inventory Tabs Hotkey. \
There is a bit of an unintended consequence to this mod where you can access the Prestige tab while in a raid, which is something you can't normally do. From my testing this didn't cause any issues but if it does for you please let me know. \
 \
As of version 1.2.0 you can also jump to a specific inventory tab with a specific hotkey, every hotkey is customizable in the F12 menu but by default they are: \
Overall: C \
Gear: G \
Health: R \
Skills: J \
Map: M \
Tasks: T \
Achievements: K \
Prestige: L
